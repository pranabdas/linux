### Installation in Ubuntu/Debian
```bash
sudo apt install -y --no-install-recommends \
    texlive-latex-recommended \
    texlive-latex-extra \
    texlive-fonts-extra \
    texlive-science \
    texlive-pictures \
    texlive-luatex
```
